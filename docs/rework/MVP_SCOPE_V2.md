# MVP Scope v2

Статус: не утверждён.

## Входит

Заполнить после утверждения `PRODUCT_V2.md`.

## Не входит

Пока по умолчанию не входит:

- аккаунты;
- облако;
- Health Connect / Google Fit;
- AI;
- монетизация;
- сторы и публичный релиз;
- перенос всех модулей v1 без пересмотра смысла.

## Условия готовности нового MVP

- Есть один главный сценарий, а не список модулей.
- Каждый экран отвечает на понятный пользовательский вопрос.
- Нет обязательной зависимости от старого Today screen.
- Локальное хранение остаётся, если не принято иное архитектурное решение.
- APK можно собрать и установить для внутренней проверки.
