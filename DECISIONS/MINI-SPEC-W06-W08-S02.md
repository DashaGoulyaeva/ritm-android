# MINI-SPEC W-06, W-08, S-02 — Water history + reminders + settings

**Статус:** Принято  
**Дата:** 2026-04-20  
**Роль:** Product Agent

## User stories

1. Как пользователь, я хочу видеть историю логов воды по дням, чтобы понимать динамику привычки.
2. Как пользователь, я хочу включать/выключать напоминание о воде и задавать время, чтобы поддерживать рутину.
3. Как пользователь, я хочу управлять дневной целью воды в настройках, чтобы цель соответствовала моему режиму.

## Acceptance criteria

### W-06 WaterHistoryScreen
1. Экран показывает записи воды/чая/кофе, сгруппированные по дням.
2. Порядок: сначала новые дни и новые записи.
3. Есть корректный empty state.
4. Экран открывается из блока воды на Today.

### W-08 WaterReminderWorker
1. Есть Worker, отправляющий локальное уведомление о воде.
2. Планировщик создаёт уникальную periodic-задачу без дублей.
3. При выключении reminder задача отменяется.
4. Время reminder читается из настроек и применяется при перепланировании.

### S-02 Water settings
1. Можно менять цель воды (в стаканах).
2. Можно включать/выключать reminder воды.
3. Можно задавать время reminder в формате `HH:mm`.
4. Изменения сохраняются в DataStore и отражаются после перезапуска.

## Edge cases

- Невалидное время reminder не сохраняется и отображает понятную ошибку.
- При отсутствии разрешения на уведомления Worker не падает (fail-safe: без crash).
- Если история пустая, экран не показывает «пустой список» без пояснения.
- При повторном сохранении reminder с тем же временем не создаются дублирующие work-записи.
