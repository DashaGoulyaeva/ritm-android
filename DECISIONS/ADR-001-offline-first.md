# ADR-001: Офлайн-first, без облачной синхронизации в MVP

**Статус:** Принято  
**Дата:** 2026-04-20

## Контекст

Приложение трекает персональные данные о здоровье. Пользователь ожидает приватности.

## Решение

Все данные хранятся только локально на устройстве. Room — единственное хранилище. DataStore — для настроек. Интернет-разрешения не запрашиваются.

## Последствия

- (+) Полная приватность. Никаких серверов, никакой утечки.
- (+) Работает без сети всегда.
- (+) Нет сложности бэкенда в MVP.
- (-) Нет резервного копирования (post-MVP).
- (-) Нет синхронизации между устройствами (post-MVP).
