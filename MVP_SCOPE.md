## ВХОДИТ в MVP

### Core
- [ ] Today screen — единый экран дня с агрегированным статусом
- [ ] Навигация между модулями
- [ ] Локальное хранение (Room + DataStore)
- [ ] Локальные уведомления (WorkManager + NotificationManager)

### Привычки
- [ ] Создание привычки (название, тип: POSITIVE/AVOID, частота)
- [ ] Архивирование привычки
- [ ] Отметка выполнения за день (DONE/SKIPPED/FAILED)
- [ ] Список активных привычек на Today
- [ ] Стрик по каждой привычке
- [ ] История выполнения (простой список/grid по дням)

### Цикл
- [ ] Отметка начала/конца менструации
- [ ] Расчёт текущего дня цикла
- [ ] Прогноз следующей менструации (на основе средней длины)
- [ ] Дневник дня: выделения, симптомы, настроение, заметка
- [ ] Календарь с отметками

### Вода
- [ ] Быстрый лог: вода / чай / кофе (стаканами)
- [ ] Дневная цель (в стаканах, настраивается)
- [ ] Прогресс на Today
- [ ] История по дням

### Fasting
- [ ] Выбор режима: 12/12, 14/10, 16/8, 18/6, 20/4, 23/1
- [ ] Старт/стоп сессии вручную
- [ ] Таймер активной сессии (на Today и на экране fasting)
- [ ] История завершённых сессий

### Уведомления
- [ ] Напоминание выпить воду (настраиваемое время)
- [ ] Уведомление о завершении окна fasting
- [ ] Напоминание отметить привычки (вечернее)
- [ ] Текст уведомлений — мягкий, с юмором

### Настройки
- [ ] Управление привычками (список + архив)
- [ ] Цель по воде (количество стаканов)
- [ ] Режим fasting по умолчанию
- [ ] Управление напоминаниями (вкл/выкл + время)

## НЕ ВХОДИТ в MVP (post-MVP)

### Аккаунты и синхронизация
- Регистрация / вход
- Облачная синхронизация
- Резервное копирование в облако

### Интеграции
- Health Connect
- Google Fit
- Экспорт данных (PDF, CSV)

### Продвинутые фичи
- AI-рекомендации
- Сложная аналитика и графики трендов
- Беременность и планирование зачатия
- Медицинские режимы и обещания
- Тёмная тема

### Монетизация и социальное
- Подписка / платные фичи
- Реклама
- Социальные функции (челленджи, друзья)
- Публичные профили

### UX
- Onboarding (пошаговый)  ← *можно добавить в конце MVP*
- Виджеты для домашнего экрана
- Wear OS
